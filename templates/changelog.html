Changelog

Website Update (Medium | 0.0.1 -> 0.1.0) 03/13/2020

Dear users, 

I'm here to announce we have a medium updates:

  1. Added Public Profile Feature - From now on, every user has a public profile image, however, user for now can't choose whether they want to display profile public or not. Make sure to contact thru survey if you don't want your profile page to be public. The link to public profile varied based on your username. Format is something like this: https://readsaltie.pythonanywhere.com/profile/<YOUR USERNAME HERE> . 
  2. Added Profile Image - Now you can have profile image displayed on your public profile page. (Still in experiment)
  3. Added Follow Feature - If you go to a user's public profile page, you can click a "follow" button, it sort like YouTube's subscribe button.
  4. Added Notification Feature - If the user that you chose to followed has published a new book, you will see on the inbox button at the top, click on the "inbox" text, and you should see the notificatoon there. Click and you can read the book.
  5. Added Publish Manually Feature - You can now write book manually using this page. It's auto-save. If you want to publish your book, you can do so by go to profile, scroll down to draft book section, and click on the book you want to publish, then upload an image for the book, and click publish button.

And yes, that's all updates we got, if you would like to suggest a feature, make sure you do so by go to /survey . Thank you and enjoy the website. :)

Website Update (Minor | 0.0.0 -> 0.0.1) 03/08/2020

Dear users, 

I'm here to announce we have a few minor updates:

  1. Added Weekly Contest - Starting today (03/08/2020) and every Sunday, there will be 2 popular books automatically selected by our system, one is most viewed and one is most rated. However, there is no opt-out feature, so if you don't want your book be put on our popular books, please contact us thru /survey.
  2. Track view for each book - Starting today (03/08/2020), every time when you clicked on a book, that book's view will increase. And this is the only information we will keep track.  Your privacy still remain private.

And yes, that's all updates we got, if you would like to suggest a feature, make sure you do so by go to /survey . Thank you and enjoy the website. :)
